──────────────────────────────

■同梱ファイル

index.html		入り口用
main.html		フレーム表示ファイル
toppage.html	左フレーム表示用
menu.html		右フレーム表示用

--右フレーム--
gallery.html	展示一覧用

--左フレーム--
sample.html		サンプル用
link.html		リンク用
memo.html		日記用
pict.html		イラスト展示用

--他--
imgフォルダ		画像が入ってます
cssフォルダ		外部CSSが入ってます

--外部CSS--
bace.css		ベースになるcss
left.css		左フレーム用
right.css		右フレーム用

外部CSSは分かりやすい用に分けてます。

──────────────────────────────

デザインを外部CSSで指定しています。
全体の文字の色や大きさ、タイトルバーの色、ラインの色…
など、変更したい場合は外部CSSをいじってみて下さい。

テンプレートは「標準準拠」モードで作成しています。
以下のドキュメント宣言↓を変更すると、デザインも変わってしまいますのでご注意下さい。
<!DOCTYPE html PUBLIC "-//W3C//DTD HTML 4.01 Transitional//EN" "http://www.w3.org/TR/html4/loose.dtd">
※HPビルダーで開くと変更されてしまうので注意。

同梱バナーは自作なので、土台にして好きにいじって下さって構いません。
